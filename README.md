# eCommerce-website

## Quick start

1. Clone this repo using:
  ```shell
  $ git clone git@github.com:Levshenkov/Ecommerce-convenience.git
  ```

2. To install dependencies and clean the git repo run:

  ```shell
  $ yarn install
  ```
  
3. Run project in Dev mode

  ```shell
  $ yarn run dev
  ```
